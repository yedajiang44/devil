# DEVIL

devil 是一个消耗计算机处理器和内存资源的命令行工具

## 使用

### 获取

下载发布版或源码编译

### 帮助信息

执行 devil 即可获取相关信息

### 处理器

```
devil cpu consume -u 50 -t 60
```

| 短选项 | 长选项 | 含义                                                                   |
| ------ | ------ | ---------------------------------------------------------------------- |
| u      | used   | 消耗掉剩余资源的百分比                                                 |
| t      | time   | 消耗的持续时间。单位：秒，默认为 60 秒，如设置小于等于 0，则无时间限制 |

### 内存

```
devil ram consume -u 50 -t 60
```

| 短选项 | 长选项 | 含义                                                                   |
| ------ | ------ | ---------------------------------------------------------------------- |
| u      | used   | 消耗掉剩余资源的百分比                                                 |
| t      | time   | 消耗的持续时间。单位：秒，默认为 60 秒，如设置小于等于 0，则无时间限制 |
